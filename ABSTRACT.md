# ABSTRACT

The **Sentinel Security Suite** is an integrated, beginner-friendly **web-based** cybersecurity educational tool designed to provide a hands-on learning experience for users interested in digital security. In an era of increasing cyber threats, understanding the mechanisms of common attacks and protection methods is crucial for both students and general internet users.

This project implements a multi-modular system that addresses five core areas of cybersecurity:
1.  **Phishing Detection**: Utilizing heuristic analysis to identify suspicious URL patterns and potential social engineering attempts.
2.  **Malware Analysis**: Providing file integrity verification through SHA-256 hashing and scanning for suspicious file extensions.
3.  **Brute Force Security**: Simulating login-based attacks to demonstrate the importance of lockout policies and password complexity.
4.  **IP Spoofing Detection**: Monitoring and identifying potential identity-based network threats.
5.  **Data Cryptography**: Implementing robust AES-based (Fernet) encryption and decryption for Documents, Audio (MP3), and Video (MP4) to illustrate the principles of data confidentiality.

Built using **Python** and the **Streamlit** framework, the suite offers a unified, interactive **web-based** interface that simplifies complex security operations. By bridging the gap between theoretical knowledge and practical application, the **Sentinel Security Suite** serves as a comprehensive educational platform that empowers users to detect, analyze, and mitigate digital risks effectively, thereby promoting a more secure browsing and computing environment.
